Antes de empezar a trabajar (en cualquiera de los dos), bájate los últimos cambios que hayas podido hacer en el otro:
git pull

Al terminar, guardas y subes a la nube:
git add .
git commit -m "11.1 Historial trabajadores y problemas Aarón"
git push origin main

Ordenar código:
Shift+Alt+F

Asegurar directorio
firebase use pedidos-rail-app-2025-87f2c

Guardar versión
firebase deploy
